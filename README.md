# FreePixa

A image searcher using Unsplash API, made with Sveltekit.

## Features

- Image searching
- Image viewer
- Infinite scrolling
- Favourites
- Light/dark mode

## Tech Stack

**Client:** Svelte & Sveltekit, Shadcn-svelte, Typescript, Tailwind
